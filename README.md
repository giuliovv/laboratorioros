# laboratorioros
Polimi ROS lab 2020

Edo robot kinematic and dynamic simulation (https://edo.cloud/)

### Display model:

```shell
 $ roslaunch edo display.launch.launch 
 ```

### Kinematic simulation:
```shell
 $ roslaunch edo kinematic_simulation.launch 
 ```
 
 ### Dynamic simulation:
```shell
 $ roslaunch edo dynamic_simulation.launch 
 ```
